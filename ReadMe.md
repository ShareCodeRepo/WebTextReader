# 📖 WebTextReader (v1.0)

* 웹 브라우저에서 텍스트 파일을 쉽고 편하게 읽을 수 있는 웹 기반 텍스트 뷰어입니다.
* 복잡한 설정 없이 파일을 불러와 바로 독서를 시작할 수 있으며, 마지막 읽던 위치를 기억합니다.

✨ 주요 기능

* 파일 로드: .txt 파일을 브라우저로 직접 불러와 읽기 기능.
* 책갈피(Bookmark): 현재 읽고 있는 위치를 로컬 스토리지에 저장하고, 
  나중에 동일한 파일을 열었을 때 해당 위치로 즉시 이동.

## 앞으로 개선할 기능

* 글자 크기 및 줄 간격: 사용자 시력과 취향에 맞는 가독성 조절.
* 진척도 표시: 현재 전체 내용 중 어느 정도 읽었는지 진행률 확인.
* LocalStorage: 사용자 설정 및 책갈피 데이터 저장

## 🛠 기술 스택

* HTML5 / CSS3: 레이아웃 및 스타일링.
* JavaScript (Vanilla JS): 파일 처리 및 리더 로직 구현
